# 小红书MCP项目

## Markdown一键转小红书爆款格式

🎯 课程作业 | 纯手动搭建 | 无脚手架

---

# 一、项目简介

- 项目名称：小红书MCP智能排版系统
- 开发工具：VS Code + HTML/CSS/JS
- 运行环境：浏览器本地运行
- 演示工具：Slidev

核心功能：

1. Markdown自动转小红书排版
2. 3种风格模板一键切换
3. 实时预览、一键复制发布

---

# 二、项目文件结构（纯手动创建）

全部文件手动新建，无自动生成，共5个文件：

- index.html 页面结构
- style.css 样式美化
- app.js 转换逻辑
- slides.md 答辩PPT
- package.json 项目配置

---

# 三、核心功能演示：界面展示

小红书风格UI，粉色主题，左右分栏设计

- 左侧：输入普通文本/Markdown
- 右侧：实时生成小红书排版
- 顶部：可切换3种模板

---

# 四、功能演示：文本转换

输入普通Markdown → 自动生成小红书格式

- 标题加emoji图标
- 列表自动美化
- 末尾自动添加流量话题标签

---

# 五、功能演示：一键复制

点击按钮 → 复制成品内容
弹窗提示“复制成功”，直接去小红书粘贴发布

---

# 六、运行与部署

1. 安装依赖：npm install
2. 启动PPT：npm run dev
3. 运行工具：打开index.html
4. 构建静态页：npm run build

---
